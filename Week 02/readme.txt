This project has been broken into 3 parts

Sean is doing the map
Ross is doing the weather
Sagarika is doing integration

To compile from command line:
Windows:
javac -cp .;./java-json.jar CSE360\Universe.java

Linux:
javac -cp .:./java-json.jar CSE360/Universe.java

To run java applet from command line:
Windows:
java -cp .;./java-json.jar CSE360.Universe

Linux:
java -cp .:./java-json.jar CSE360.Universe
